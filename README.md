# Linux Lab Scripts

Колекція Bash-скриптів для лабораторних робіт
з курсу "Системне програмування та адміністрування ОС".

## Вміст

- count_files.sh - скрипт для підрахунку файлів у /etc

## Використання

./count_files.sh

## Лабораторна робота №4
Створення RPM-пакета

У папці `packaging/rpm` розміщено spec-файл:
- `count-files.spec` — файл для збірки RPM-пакета скрипта `count_files.sh`.
RPM-пакет було зібрано локально, перевірено встановлення, запуск та видалення.
